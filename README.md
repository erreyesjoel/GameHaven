## 🎯 Objetivo

Este proyecto tiene fines **educativos y de aprendizaje**, y consiste en **dos partes principales**:  

1. **Página de gestión interna**  
   - Construida con **Django**, usando **templates, Jinja, static files, HTML, Sass y JavaScript**.  
   - Permite administrar los datos de la aplicación de forma segura y organizada.

2. **Frontend para el cliente**  
   - Construido con **Angular**, usando **TypeScript, Sass y HTML**.  
   - Consume las **APIs del backend Django** para mostrar los datos a los usuarios finales.  

- La base de datos será **PostgreSQL**.  
- Se usa **Git** para control de versiones y seguimiento del proyecto.  

---

💡 Ideal para mostrar en portfolio cómo integrar **backend Django con frontend Angular** y cómo organizar un proyecto web completo para aprendizaje.
